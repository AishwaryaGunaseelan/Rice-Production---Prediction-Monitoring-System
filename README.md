# Rice Production Prediction & Monitoring System

This project implements an end-to-end machine learning pipeline for predicting rice production using web-scraped agricultural data.

## Project Highlights

- Automated data collection via web scraping  
- Data preprocessing and cleaning  
- Ridge Regression modeling  
- Hyperparameter tuning using Optuna  
- Experiment tracking with MLflow  
- Model packaging (.pkl)  
- FastAPI inference API  
- Drift detection & monitoring logic  
- Retraining strategy  
- Dockerization  
- CI/CD workflow  

## Objective

To demonstrate applied machine learning and MLOps practices in an agriculture-related prediction problem.

## Tech Stack

Python, Pandas, Scikit-learn, Optuna, MLflow, FastAPI, Docker
